About

This project provides a clean and interactive dashboard for tracking the top cryptocurrencies in real time. Users can browse the top 100 coins, search by name or symbol, and mark selected coins as Favorites (up to 5). For each coin, the application displays detailed price information and allows opening a dedicated chart view that presents historical candlestick data and live price updates, similar to common trading platforms. The project demonstrates working with external APIs, dynamic DOM rendering, UI navigation between views, and state management within a TypeScript-based application.
